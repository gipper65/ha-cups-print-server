# Home Assistant CUPS Print Server Add-on Repository

This repository contains a custom CUPS printing add-on for Home Assistant.

## Installation

1. Add this repository to Home Assistant:
   - Go to **Settings > Add-ons > Add-on Store**
   - Click the **â‹®** menu (top right)
   - Select **Repositories**
   - Add: `https://github.com/gipper65/ha-cups-print-server`

2. Install the CUPS Print Server add-on from the store

3. Configure and start the add-on

## Add-ons

### CUPS Print Server

Full-featured CUPS printing system with REST API for Home Assistant.

**Features:**
- Network printer support
- REST API for printing from automations
- Persistent configuration
- Web interface for printer management
- Epson and Gutenprint drivers included

See the [add-on README](cups-print-server/README.md) for detailed documentation.
